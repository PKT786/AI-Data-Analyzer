# 📊 AI Data Analyzer

An AI-powered data analysis application that automatically analyzes Excel and CSV files, generates insights, and creates interactive visualizations.

## 🚀 Features

✅ Upload Excel / CSV files

✅ AI-powered data analysis

✅ Natural language questions

✅ Automatic chart generation

✅ Business insights

✅ Interactive visualizations

✅ Data-driven recommendations


## 🛠️ Technologies Used

- Python

- Streamlit

- OpenAI API

- Pandas

- Plotly

- Excel Processing
